Class and Static Keyword

Create object using class

Javascript classes can also have static methods

Static methods can be created using static keyword

        
            class Employee{
                constructor(empId, empName, empAge){
                    this.id = empId;
                    this.name = empName;
                    this.age = empAge;
                }
        
                swipeIn(){
                    console.log('Employee id',this.id,' has swiped in at', new Date().toDateString());
                }
        
                static code(){
                    console.log('Employee is coding..')
                }
            }
        
            let empObj = new Employee(102, "John", 30);
            console.log('Id -',empObj.id);
            console.log('Name -',empObj['name']);
        
            empObj.swipeIn();
        
            // Call static method directly by class name
            Employee.code();

Output

        
            Id - 102
            Name - John
            Employee id 102  has swiped in at Tue Dec 10 2024
            Employee is coding..

Class Inheritance

Create a sub-class using the extends keyword

Access the inherited properties and methods from child class

Use super keyword to call parent class properties

        
            class PartTimeEmployee extends Employee{
        
                constructor(empid, empname, empage, contractPeriod){
                    super(empid, empname, empage);
                    this.contractPeriod = contractPeriod;
                }
        
                swipeIn(){
                    console.log('Contract Employee Swiped In');
                    super.swipeIn();
                }
        
            }
        
            let partTimeEmpObj = new PartTimeEmployee(101, "Sam", 29, 3)
        
            PartTimeEmployee.code();
            partTimeEmpObj.swipeIn();

Output

        
            Employee is coding..
            Contract Employee Swiped In
            Employee id 102  has swiped in at Tue Dec 10 2024
