Error Handling and Error object

Try and Catch

The try statement defines a code block to run (to try).

The catch statement defines a code block to handle any error.

Error of different type can be handled differently inside the catch block

Build-in error objects

- Eval error
- Internal error
- Range error
- Reference error
- Syntax error
- Type error
- URI error

Syntax

        
            try {
                Block of code to try
              }
              catch(err) {
                Block of code to handle errors
              }
        
    
        
            function demo(){
                try{
                    let bike = newbike;
                }catch(error){
                    if(error instanceof ReferenceError){
                        console.log("Reference Error occured : ", error.message);
                    }else if(error instanceof TypeError){
                        console.log("Type Error occured : ",error.message)
                    }else{
                        console.log("Some Error occured : ",error.message)
                    }
                }
            }
        
            demo();
            console.log("Test")

Output

        
            Reference Error occured :  newbike is not defined
            Test

Finally and Throw

The finally statement lets you execute code, after try and catch, regardless of the result.

The throw statement defines a custom error.

Syntax

        
            try {
                Block of code to try
            }
            catch(err) {
                Block of code to handle errors
            }
            finally {
                Block of code to be executed regardless of the try / catch result
            }
        
    
        
            function demo(){
                try{
                    let bike = null;
                    if(bike === null){
                        throw new Error("Bike can not be empty");
                    }
                }catch(error){
                    if(error instanceof ReferenceError){
                        console.log("Reference Error occured : ", error.message);
                    }else if(error instanceof TypeError){
                        console.log("Type Error occured : ",error.message)
                    }else{
                        console.log("Some Error occured : ",error.message)
                    }
                }finally{
                    console.log("Finally...")
                }
            }
        
            demo();
            console.log("Test");

Output

        
            Some Error occured :  Bike can not be empty
            Finally...
            Test
