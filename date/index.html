Date Object and Its methods

Date object created using new operator points to current system date and time stamp.

Same information can be printed in milliseconds using getTime() method.

Date usually contains month, year and date. These can be extracted using getDate(), getMonth() and getFullYear().

To print the date object in the string format we can use toLocaleString() and toLocaleDateString().
toLocaleString() - Prints both date and time stamp information in string format.
toLocaleDateString() - Prints only the date information in string format. 

        
            let dateOne = new Date();
            console.log("Date One : ", dateOne);
        
            console.log("In milliseconds", dateOne.getTime());
        
            console.log("Date : ", dateOne.getDate());
            console.log("Month :", dateOne.getMonth());
            console.log("Year : ", dateOne.getFullYear());
        
            dateOne.setFullYear(2050, 10, 7);
            console.log("After setting date information : ", dateOne);
        
            let today = new Date();
        
            let dateDiff = ( dateOne - today ) / (1000*60*60*24);
            console.log("Difference days : ", dateDiff);
            console.log("Difference days (Actual days): ", Math.floor(dateDiff));
        
            console.log("Today date with time in string", today.toLocaleString());
            console.log("Only date in string format", today.toLocaleDateString())
        
            let newDate = new Date('2025-04-24');
            console.log("New Date : ", newDate)
