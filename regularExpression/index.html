Regular Expression(RegEx)

The RegExp object can be constructed using either of the two ways
- using RegExp constructor (let pattern = new RegExp(pattern, modifiers))
- as a literal value by enclosing within forward-slash(/) ( let pattern = /pattern/modifiers );

Meta Characters are ones which has special meaning regEx.

Quantifiers indicates the frequency of the character that can be matched using regEx.
Some quantifiers are +(one or more), *(zero or more), ?.

Create RegExp for email validation :
- Valid Email - username@email.com username@email.com
- Diffrent Email - test@email.com [a-z]@email.com(It will match only one character) => [a-z]+@email.com
- Different Domain - demo@mail.com [a-z]+@[a-z]+.com
- Numbers in Email - test123@email.com [a-z0-9]+@[a-z]+.com
- Underscore in email - test_123@email.com [a-z0-9_]+@[a-z]+.com
- .in in last - test12@email.in [a-z0-9_]+@[a-z]+.(com|in)
- When adding $ in place of @ - its invalid email but its showing valid email - test$mail.com - [a-z0-9_]+@[a-z]+\.(com|in)
- Invalid Id (starts with special character) - !abc@mail.com - ^[a-z0-9_]+@[a-z]+\.(com|in)
- Ends with coma(have extra 'a') - abc@mail.coma - ^[a-z0-9_]+@[a-z]+\.(com|in)$
- Include Capital alphabets -^[a-zA-Z0-9_]+@[a-z]+\.(com|in)$
- Replace by word meta character -^[\w]+@[a-z]+\.(com|in)$

exec() - If pattern matched it returns array otherwise null.

test() - If pattern matched it returns true otherwise false.

search() - match string btn RegEx and String.

match() - match RegEx in given string.

replace() - find and replace the matched string with new substring

        
            let regEx1 = new RegExp(/^[\w]+@[a-z]+\.(com|in)$/, "i");
   
            console.log(" RegExp- exec() method",regEx1.exec('username@gmail.com'), regEx1.exec('!username@gmail.com'));
            console.log("RegExp - test() method", regEx1.test('username@gmail.com'), regEx1.test('usernamegmail.com'));

            let regEx2 = /^[\w]+@[a-z]+\.(com|in)$/;
        
            console.log("exec() method",regEx2.exec('username@gmail.com'), regEx2.exec('!username@gmail.com'));
            console.log("test() method", regEx2.test('username@gmail.com'), regEx2.test('usernamegmail.com'));


            let str = "HelloWorld";
            console.log("search() method", str.search(/ello/), str.search(/Well/))
            console.log("match() method", str.match(/oWo/), str.match(/Hi/));

            // It will replace only first occurence
            console.log("replace() method", str.replace(/l/,'o'));

            // To replace all the occurences we need to specify flag 'g'
            console.log("Replace all occurences", str.replace(/l/g,'o'))
