String and Its functions

Access the properties of object by for loop

length - Length is a read-only property that returns the length of the given string.

indexOf() - Used for finding the index of a character or a string present inside the string variable

charAt() - If the index is known and we want to find the character at that position it can be done using charAt().

split() - To split the string based on certain character we can use the split() method.
The split() method splits a string into an array of substrings.
Syntax - string.split(separator, limit)

substr() - To extract some characters from the existing string we can use the substr() method.
The substr() method extracts a part of a string.
Syntax - string.substr(start, length)

substring() - To extract characters from a string using the starting and ending index use the substring() method.
The substring() method extracts characters from start to end (exclusiv). (end index value = end-1)
Syntax :- string.substring(start, end) 

        
            let myString = "Aeroplane";
            console.log("String Length is : "+myString.length);
            console.log("Get first character of string", myString[0]);

            let findIndex = myString.indexOf('plane');
            let findIndexNew = myString.indexOf('water');
            console.log('Index value : ', findIndex, findIndexNew);

            let char = myString.charAt(6);
            console.log("Character at 6 index : ",char);

            let splitStr = myString.split('e');
            console.log("Splitted array : ", splitStr, myString.split('e',2) , ("How are you doing today").split(" "));

            let substrString = myString.substr(0,4);
            console.log("Substr : ", substrString, myString.substr(3,5), myString.substr(2));

            let substring = myString.substring(0,5);
            console.log("Substring : ", substring, myString.substring(3,5), myString.substring(2));

            let upperStr = myString.toUpperCase();
            console.log("toUpperCase : ",upperStr);

            let lowerStr = myString.toLowerCase();
            console.log("toLowerCase : ", lowerStr);
