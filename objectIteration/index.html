Object Iteration

Access the properties of object by for loop

Before ES6 it was the way of iteration object

        
            let employee = { id: 1001, name: 'John', age: 32}

            for(let key in employee){
                console.log(key, ":", employee[key]);
            }

Output

        
            id : 1001
            name : John
            age : 32

After ES6 we can iterate object by using multiple ways

First we will convert object into array using:

Object.keys() - This methos creates an array that contains all the keys

Object.values() - This methos creates an array that contains values for every key

Object.entries() - This methos creates an array of arrays

Example : [ [key,value], [key,value], ...[key,value] ] 

        
            let employee = { id: 1001, name: 'John', age: 32}

            console.log("Object iteration using Object.keys")
            Object.keys(employee).forEach((key) => {
                console.log(key, ":", employee[key]);
            });

            console.log("Object iteration using Object.values")
            Object.values(employee).forEach((values) => {
                console.log("Values of emp object are", values);
            });

            console.log("Object iteration using Object.entries")
            Object.entries(employee).forEach((arr) => {
                console.log(arr[0], "----", arr[1]);
            });

Output

        
            Object iteration using Object.keys
            id : 1001
            name : John
            age : 32
            Object iteration using Object.values
            Values of emp object are 1001
            Values of emp object are John
            Values of emp object are 32
            Object iteration using Object.entries
            id ---- 1001
            name ---- John
            age ---- 32

Freeze the object - We can not add new property and modify existing property.

Seal the object - Modifying an existing property is allowed but Adding new properties is not allowed

        
            const userObj = {
                name: 'Nane',
                email: 'nane@yopmail.com',

            Object.seal(userObj);
